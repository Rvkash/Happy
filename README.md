## ⌨ Como executar o projeto
```bash
# Clonar o repositório
git clone https://github.com/Rvkash/Happy
# Entrar no diretório
cd Happy
# Baixar as dependências
npm install
# Executar o servidor
npm start
```
Feito isso, abra o seu navegador e acesse `http://localhost:5500/`
